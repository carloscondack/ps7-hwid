# CCIT Tech Tools

A collection of IT automation scripts for enterprise environments, served at [tools.ccittech.solutions](https://tools.ccittech.solutions).

## Tools

| Folder | Description |
|---|---|
| [`/intune`](./intune/) | Intune Autopilot enrollment scripts for Windows devices |

## Contributing

Add new scripts under a folder named for the technology or platform (e.g. `/intune`, `/azure`, `/windows`, `/network`). Include a `README.md` in each folder documenting usage.
